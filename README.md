# 🏥 MediBridge – Smart Doctor Finder & Appointment Booking App

MediBridge is a lightweight Android application designed to help users quickly find doctors and book appointments without requiring authentication. The system focuses on usability, fast search performance, and accessible healthcare discovery through a clean and intuitive mobile interface.

---

## 📱 Features

- Browse doctors by name, specialization, or hospital  
- Search and filter by city, department, or hospital  
- View complete doctor profiles including schedules and details  
- Book appointments directly from the profile  
- View upcoming appointments in a structured dashboard  
- Lightweight design with fast navigation  
- No login required for quick access

| Backend       | Firebase Firestore (real-time database) |
| Storage       | Imgur (doctor images) |
| Image Loading | Glide / Coil |

---

## 🛠 Setup Instructions

1. Clone this repository  
2. Create a Firebase project and enable Firestore and Storage  
3. Download `google-services.json` and place it in the `app/` directory  
4. Add sample doctor and appointment data into Firestore  
5. Build and run using Android Studio
